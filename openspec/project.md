# Project Context

## Purpose
Android technical challenge app to demonstrate senior-level architecture,
Clean Architecture, MVVM, Jetpack Compose, and testing.

## Tech Stack
- Kotlin
- Android SDK 34+
- Jetpack Compose
- Clean Architecture
- MVVM
- Retrofit
- Coroutines
- Hilt

## Constraints
- No authentication
- No purchases
- No over-engineering
- Interview-defensible decisions

## Dependency Management

- All libraries MUST use latest stable versions.
- Version Catalog (libs.versions.toml) is mandatory.
- No alpha or deprecated libraries unless justified.
